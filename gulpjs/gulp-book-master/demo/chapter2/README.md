## 安装依赖
打开命令行切换至此目录后，在命令行输入
```
npm install
```

将会安装此示例所需的模块

-------

若没有安装模块或安装模块失败。在命令行运行 `gulp`。则会出现如下报错：
```
module.js:338
    throw err;
          ^
Error: Cannot find module 'gulp-xxx'
```
-------
文件夹中会多出一个dist文件，里面有对应的压缩过后的文件夹。 
具体对应的书写步骤自己可以相关的查看对应的目录下面的文件夹。
